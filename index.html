Serps - Home

The PHP Search Engine Result Page Scraper

Web scraping (web harvesting or web data extraction) is a computer software technique of extracting information from websites. Usually, such software programs simulate human exploration of the World Wide Web by either implementing low-level Hypertext Transfer Protocol (HTTP), or embedding a fully-fledged web browser, such as Mozilla Firefox.

Wikipedia

What is it?

Serps is a set of tools that ease the parsing of popular search engines. It helps to parse SERP (Search Engine Result Page) and gives you a standard output of what is parsed.

The problem

Most of times search engines don't want you to parse them, and they don't offer a documentation or a standard way to extract the results from the SERP.

The solution

We tried to solve this problems by analysing how search engines behave and we built the necessary tools to work with them, from the URL generation to the parsing of the results. At the endpoint we offer a standard and documented API.
